# Local Hosting

## Быстрый запуск

### Только локально
1. Дважды нажмите `quick_start_local.cmd`.
2. Сайт откроется по адресу `http://127.0.0.1:5000`.

### Локально + публичная ссылка
1. Дважды нажмите `quick_start_public.cmd`.
2. Откроются два окна PowerShell:
   - локальный сервер
   - публичный туннель
3. Во втором окне появится внешняя ссылка от `localhost.run`.
4. Пока оба окна открыты, сайт доступен из интернета.

## Ручной запуск

### Локальный сервер
```powershell
.\start_local_server.ps1
```

### Публичный туннель
```powershell
.\start_public_tunnel.ps1
```

## Что важно

- Для публичного доступа компьютер должен быть включён.
- Для работы публичной ссылки локальный сервер тоже должен быть запущен.
- Если туннель перезапустить, ссылка может измениться.
- Публичная ссылка сейчас создаётся через `localhost.run`.
- Если PowerShell блокирует запуск скрипта, используйте `.cmd` файлы из этого проекта.
